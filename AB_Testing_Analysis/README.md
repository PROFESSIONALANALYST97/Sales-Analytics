# A/B Test Analysis: Website Conversion Optimization

## Project Overview
This project determines whether website modifications led to a statistically significant increase in conversion rates. 

## Workflow & Features
* **Statistical Analysis:** Applied `scipy.stats` to calculate P-Values and verify the Null Hypothesis.
* **Interactive Dashboard:** [View Analysis on Tableau Public](https://public.tableau.com/app/profile/jan.cwirko/viz/PortfolioProjectSalesAnalysis/StatisticalInsights)
* **Note:** Use the **"Statistical Insights"** tab in the Tableau link to see the A/B test results.
* **Visual Preview:**
![Dashboard Preview](Dashboard_Overview.png)

## Final Recommendation
Based on the lack of statistical significance, the recommendation was to iterate on the test design or explore different hypotheses for the identified stable segments.
